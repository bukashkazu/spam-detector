# spam-detector
Сервис в виде чата для автоматического обнаружения спама, фишинга и опасных ссылок.

## Концепция

Сервис анализирует входящие сообщения (текст + ссылки) и возвращает:
- Вердикт: нормальное сообщение, спам или фишинг
- Вероятность классификации  
- Список подозрительных элементов (ссылки, ключевые слова)

## Возможности

- Анализ текста и ссылок в реальном времени
- Объяснимые результаты с указанием причин
- Авторизация по email с одноразовыми кодами
- Возможность добавления своих признаков

## Авторизация
1. Ввод email и пароля
2. Получение одноразового кода (действует 5 минут)
